# NHAI Toll Plaza Management System

A real-world Indian highway toll plaza dashboard built with React, Vite, Tailwind CSS, React Router, and localStorage.

## Features

- All India toll plaza dashboard
- State-wise toll plaza filtering
- Search by city, state, and highway
- Vehicle entry with Indian vehicle number format
- FASTag payment simulation
- Cash and UPI payment flow
- Downloadable and printable toll receipt
- Admin dashboard with revenue and payment statistics
- Vehicle transaction history saved in localStorage
- Fully responsive desktop and mobile UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- React Router
- localStorage

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in the browser.
